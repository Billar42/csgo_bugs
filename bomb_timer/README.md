## New: Bomb Timer

### Steps
1.Install [Node js](https://nodejs.org/).

2.Download [bomb_timer](https://github.com/Billar42/csgo_bugs/tree/master/netcon_stuff/bomb_timer) script.

3.Run CSGO with `-netconport 2121`.

4.Run `bomb_timer.js`.
